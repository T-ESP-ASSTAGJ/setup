# Jamly — Annexes

## Justification des choix techniques

Notre application mobile est développée en **SwiftUI (iOS natif)**. Le site vitrine est développé en **Next.js** — framework web dédié, performant, bien maîtrisé par l'équipe, aligné avec les pratiques des grandes plateformes (Spotify, TikTok, Meta utilisent React/Next pour le web, pas React Native Web).

Next.js apporte : routing, SSR, SSG, SEO intégrés, productivité et architecture propre.
